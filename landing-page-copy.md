# Section 3: What It Does

## 1) Flags at-risk matters before they become urgent

A live Portfolio Risk Board shows:
- Matters inside 45 / 30 / 14-day windows
- Cases blocked by missing evidence
- Cases waiting on sign-off
- Matters without a recent client update

**Core metric: At-Risk Deadline Count**  
How many active matters are inside a critical window and still not submission-ready.

## 2) Turns evidence collection into a controlled workflow

Instead of chasing documents across email, WhatsApp, and folders, the system runs a structured evidence loop:

**Request -> Upload -> Check -> Fix -> Verify**

You can see, per matter:
- What is missing
- What needs correction
- What has been verified
- What is blocking submission

## 3) Blocks unsafe submissions

A matter does not move forward just because it feels "mostly there."

Submission readiness checks:
- Evidence completeness
- Outstanding blockers
- Internal review / sign-off
- Timing validity
- Required client approvals

If the case is not safe, it stays blocked.

## 4) Stops waiting-state cases from going silent

After submission, the system tracks:
- Biometrics / ID steps
- Further information requests
- Waiting-state matters
- Overdue client updates

That means fewer chasers, clearer supervision, and less reputational drift.

## 5) Creates a full proof trail

Every important case action is logged:
- Readiness changes
- Escalations
- Sign-offs
- Key updates
- Submission events

So if something goes wrong, your team can see exactly what happened and when.

# Section 8: FAQ

## Is this a practice management replacement?
No. It is an operations control layer focused on deadline risk, readiness, and visibility.

## Does this provide legal advice?
No. Legal judgment remains with the firm. The system supports operational control, workflow discipline, and escalation.

## Will this create more admin?
It should reduce admin by structuring evidence collection, surfacing blockers earlier, and reducing avoidable case-chasing.

## What case types does it support first?
V1 is focused on in-country extension and ILR workflows, starting with Skilled Worker and Partner / spouse matters.

## What if we already use case management software?
That is normal. The product is built for firms that already have tools but still lack control at the points where cases fail.
